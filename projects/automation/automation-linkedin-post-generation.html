AI Automation Case Study

LinkedIn Post Generation

A form-driven AI workflow that turns user inputs into a polished LinkedIn caption and matching generated image asset for faster social publishing.

Workflow Type: n8n Automation Category: AI Systems Format: Demo Project

Overview

A form-driven AI workflow that turns user inputs into a polished LinkedIn caption and matching generated image asset for faster social publishing.

Problem

Creating a strong social post with both copy and visuals usually requires several manual tools and creative steps.

Solution

I built an n8n workflow that starts from a form submission, generates a post draft, creates an image prompt, produces the visual asset, converts the file, and sends the final content for publishing or review.

Tools Used

n8n, GPT-4.1, Tavily, Image Generation API, Gmail

Workflow Screenshot

Key Features

  • Form-based content intake
  • AI caption generation
  • Image prompt generation
  • Matching visual asset output

Business Use Case

Great for creators, founders, and marketers who want quicker campaign turnaround and repeatable content production workflows.
