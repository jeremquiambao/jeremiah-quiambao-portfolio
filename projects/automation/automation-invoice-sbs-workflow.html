Invoice SBS Workflow

A document automation workflow that detects new invoice files, extracts their contents, logs structured data into a sheet, and sends a summary email automatically.

Workflow Type: n8n Automation Category: AI Systems Format: Demo Project

Overview

A document automation workflow that detects new invoice files, extracts their contents, logs structured data into a sheet, and sends a summary email automatically.

Problem

Manual invoice processing can be repetitive, error-prone, and slow when teams have to open files, copy details, update records, and notify stakeholders.

Solution

I designed an n8n workflow that watches Google Drive, downloads new files, extracts invoice information, appends the data to a database sheet, and generates an email summary for visibility.

Tools Used

n8n, Google Drive, Information Extractor, Google Sheets, Gmail, OpenAI

Workflow Screenshot

Key Features

  • Google Drive file trigger
  • Automated invoice data extraction
  • Database logging to sheets
  • Email summary generation

Business Use Case

Best for finance admins and operations teams that want quicker document processing, better tracking, and less repetitive data entry.
