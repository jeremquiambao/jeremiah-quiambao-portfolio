Workflow Type: n8n Automation Category: AI Systems Format: Demo Project

Overview

A retrieval-augmented workflow that ingests files from Google Drive, creates embeddings, stores them in Pinecone, and powers a chatbot that answers questions using business documents as context.

Problem

Teams often lose time searching scattered docs, SOPs, and files when answering repeated internal or customer questions.

Solution

I built an n8n workflow that watches Google Drive for new files, prepares chunks, stores vector embeddings in Pinecone, and lets an AI agent retrieve relevant context before answering each chat message.

Tools Used

n8n, OpenAI, Pinecone, Google Drive, AI Agent, Embeddings

Workflow Screenshot

Key Features

  • Automated file ingestion from Google Drive
  • Embedding-based document retrieval
  • Context-aware chatbot responses
  • Scalable knowledge base updates

Business Use Case

Useful for internal knowledge assistants, onboarding support, SOP search, and client-facing support bots that need accurate document-backed answers.
