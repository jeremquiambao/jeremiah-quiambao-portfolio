AI Automation Case Study

LinkedIn Content Creator

A scheduled content system that pulls topic data, researches with Tavily, writes post drafts with AI, and updates Google Sheets for organized content operations.

Type: n8n Automation Category: AI Systems Format: Demo Project

Overview

A scheduled content system that pulls topic data, researches with Tavily, writes post drafts with AI, and updates Google Sheets for organized content operations.

Problem

Consistent content creation is time-consuming when research, writing, and tracking all happen manually.

Solution

I built a scheduled n8n workflow that reads planned topics from Google Sheets, enriches them with Tavily research, generates LinkedIn post drafts with AI, and writes the finished output back into the sheet.

Tools Used

n8n, Google Sheets, Tavily, OpenAI, Scheduler

Workflow Screenshot

Key Features

  • Scheduled topic processing
  • Research enrichment with Tavily
  • AI-generated LinkedIn post drafts
  • Google Sheets status tracking

Business Use Case

Helpful for personal brands, consultants, and marketing teams that want a lightweight AI-assisted content engine without manual back-and-forth.
